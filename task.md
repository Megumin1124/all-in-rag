# 项目名称

Interview Prep GraphRAG System

---

# 一、项目目标

构建一个面向求职学生的智能学习系统，支持：

- 八股文知识的自动生成与管理

- 多来源数据导入（PDF / Markdown / 图片 OCR）

- 基于图结构的 GraphRAG 检索与推理

- 多模态数据处理

- 个性化学习与记忆追踪

- 面试模拟与问答系统

---

# 二、核心模块

## 1. 数据接入层

支持以下输入方式：

- PDF 文件解析

- Markdown / 文本导入

- 图片 OCR（截图面经）

- LLM 批量生成八股文

处理流程：

1. 文档解析

2. 文本清洗

3. 分块（chunking）

4. 元数据标注（来源 / 时间 / 类型）

---

## 2. 向量处理模块

- 使用 embedding 模型生成向量

- 存储至 Milvus

- 支持批量与增量更新

功能：

- 相似度检索

- 去重检测

---

## 3. 知识图谱构建（Neo4j）

定义节点：

- Concept（知识点）

- Question（问题）

- Answer（答案）

- Source（来源）

定义关系：

- Question -> Concept

- Answer -> Question

- Concept -> Concept（关联）

流程：

1. 实体抽取（NER）

2. 关系抽取

3. 图构建

4. 图更新（增量）

---

## 4. 检索系统（Hybrid + GraphRAG）

实现三层检索：

1. 向量检索（Milvus）

2. 关键词检索（BM25）

3. 图扩展（Neo4j）

融合策略：

- 合并候选集

- rerank 排序

- 构建上下文

---

## 5. 多模态处理

- 图片 → OCR → 文本

- PDF → 文本

- 所有数据统一转为：
  
  文本 + embedding + 图结构

---

## 6. 问答系统

支持：

- 单轮问答

- 多轮对话

- 面试模拟（追问）

能力：

- 基于 GraphRAG 生成答案

- 上下文记忆

---

## 7. 学习系统

功能：

- 学习进度记录

- 熟练度评分（0-5）

- 复习调度（基于遗忘曲线）

模式：

- 顺序学习

- 随机刷题

- 按公司刷题

---

## 8. 数据质量控制

- 去重（embedding + hash）

- 数据来源标记

- LLM 自动修正

- 置信度评分

---

## 9. 增量更新系统

新增数据自动：

1. 解析

2. embedding

3. 写入 Milvus

4. 构建 Neo4j 节点与关系

---

# 三、非功能需求

- 高扩展性（支持大规模数据）

- 低延迟检索

- 可持续更新

- 模块化架构

---

# 四、可选增强功能

- 热门面试题分析

- 公司题库统计

- 知识点覆盖率分析

- 错题本机制

---
